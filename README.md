## 🏭 Warehouse Management ERP System

A full-featured, production-ready Warehouse Management ERP frontend built
with modern React ecosystem tools. Designed for warehouse managers and
logistics teams to track inventory, manage orders, visualize stock levels,
and monitor warehouse zones — all in a fast, responsive, and beautiful UI.

## ✨ Features

- 📊 Real-time Analytics Dashboard with KPI cards and charts
- 📦 Inventory Management with stock level tracking
- 🛒 Order Management with status timeline
- 🗺️ Warehouse Zone & Rack Mapping
- 🚚 Supplier Management
- 📈 Reports & Export
- 🌙 Dark Sidebar with collapsible navigation
- ⚡ Fast & smooth UI with Framer Motion animations

## 🛠️ Tech Stack

- **Framework:** React 18 + Vite 5
- **Styling:** Tailwind CSS
- **State:** Zustand
- **Server State:** TanStack React Query
- **Routing:** React Router v6
- **Tables:** TanStack Table
- **Charts:** Recharts
- **Forms:** React Hook Form
- **Animations:** Framer Motion
- **Icons:** Lucide React
